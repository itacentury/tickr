# Tickr

A minimal, fast todo list app with real-time sync across devices.

**Features:** Multiple lists, offline support, undo actions, PWA installable

**Stack:** FastAPI + SQLite backend, vanilla JavaScript frontend

## Quick Start

### Docker (recommended)

Use the `docker-compose.yml` from this repository:

```bash
docker compose pull
docker compose up -d
```

### Local

```bash
pip install -r requirements.txt
python main.py
```

Open [http://localhost:8000](http://localhost:8000)

## API

Interactive docs available at `/docs` (Swagger) and `/redoc`
